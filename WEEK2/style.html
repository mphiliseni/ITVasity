Selectors

CSS selectors identify specific HTML elements as targets for CSS styles. This topic covers how CSS selectors target HTML elements. Selectors use a wide range of over 50 selection methods oﬀered by the CSS language, including elements, classes, IDs, pseudo-elements and pseudo-classes, and patterns.

Style sheet contain one or more CSS Rules

A CSS rule consists of a selector and a declaration block. The selector is the HTML element that you want to style, and the declaration block contains one or more declarations that specify the styles to apply to that element.

Shaded Yellow Class

This is a paragraph of text that is inside a div with a class of "shadedYellow". The background color of this div is light yellow, and the text color is dark yellow. The font size is 16 pixels, and the font family is Arial, sans-serif. The text is aligned to the left, and there is a 20-pixel margin around the div. The padding inside the div is 10 pixels, and the border is 1 pixel solid black. The border radius is 5 pixels, which gives the div rounded corners.

The shadedYellow class is used to apply a light yellow background color to the div, while the shaded class is used to apply a light gray background color. The text color is dark yellow for the shadedYellow class and dark gray for the shaded class. The font size, font family, text alignment, margin, padding, border, and border radius are all the same for both classes.

Class Name Selectors

Class name selectors are used to select elements based on their class attribute. A class name selector is preceded by a period (.) in the CSS rule. For example, the following CSS rule selects all elements with the class name "shaded": 

            .shaded {
                background-color: lightgray;
                color: darkgray;
                font-size: 16px;
                font-family: Arial, sans-serif;
                text-align: left;
                margin: 20px;
                padding: 10px;
                border: 1px solid black;
                border-radius: 5px;
            }

This rule applies a light gray background color, dark gray text color, 16-pixel font size, Arial font family, left-aligned text, 20-pixel margin, 10-pixel padding, 1-pixel solid black border, and 5-pixel border radius to all elements with the class name "shaded".

Id Selectors

Id selectors are used to select elements based on their id attribute. An id selector is preceded by a hash symbol (#) in the CSS rule. For example, the following CSS rule selects the element with the id "myDiv": 

            #myDiv {
                background-color: lightblue;
                color: darkblue;
                font-size: 16px;
                font-family: Arial, sans-serif;
                text-align: left;
                margin: 20px;
                padding: 10px;
                border: 1px solid black;
                border-radius: 5px;
            }

This rule applies a light blue background color, dark blue text color, 16-pixel font size, Arial font family, left-aligned text, 20-pixel margin, 10-pixel padding, 1-pixel solid black border, and 5-pixel border radius to the element with the id "myDiv".
