Welcome to CSS Week 2

CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript. It enables the separation of presentation and content, including layout, colors, and fonts. CSS is designed to enable the separation of document content from document presentation, including layout, colors, and fonts.

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in HTML or XML. It controls the layout of multiple web pages all at once.
