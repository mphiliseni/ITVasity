About this level


In this level, you will be introduced to the world of App Development. Take your first steps and learn how to craft apps using HTML,
and beautify them using CSS. By the end of this level, you will build 2 complete apps from the ground up.

You will also learn valuable skills that every software developers should have:

  • App Strategies
  • GitHub and Collaboration
  • Agile Methodology
  • The Software Development Live Cycle (SDLC)
  • Principles of UX Design
